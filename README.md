#  Bollywood Weekender App
Programmeerproject | Puja Chandrikasingh | 11059842

De Bollywood Weekender app is speciaal gemaakt voor iedereen die naar het evenement gaat. Tijdens het evenement draaien er drie podia simultaan met optredens van artiesten en DJ's en natuurlijk heel veel workshops en wedstrijden. In de app worden deze gegevens per locatie weergeven op een interactieve plattegrond. Daarnaast worden ook de laatste updates weergeven. Verder kan er ook gezocht worden op activiteiten.
